# ==========================================
# Online Banking System - System Requirements
# ==========================================

# 1. Core Runtime Environment
Java SE Development Kit (JDK) >= 17
Apache Tomcat Server >= 10.1.0

# 2. Database Server
MySQL Server >= 8.0 (Or XAMPP with MySQL module)

# 3. Build & Dependency Management
Apache Maven >= 3.8.0

# 4. Required Dependencies (Managed automatically by Maven pom.xml)
jakarta.servlet-api (6.0.0)
jakarta.servlet.jsp-api (3.1.1)
jakarta.servlet.jsp.jstl (3.0.1)
mysql-connector-j (8.4.0)
openpdf (1.3.39)

# 5. IDE (Recommended)
Eclipse IDE for Enterprise Java and Web Developers
